# International SEO Monitoring

PlayPoint's international SEO pages are grouped by language and intent. This note keeps weekly checks short while preserving the details needed to judge whether `/en/`, `/ko/`, and `/tw/` are gaining search visibility.

## URL groups

- `/en/`: English calculator shortcuts and English Play Points guides.
- `/ko/`: Korean calculator shortcuts and Korean Play Points guides.
- `/tw/`: Traditional Chinese calculator shortcuts and Traditional Chinese Play Points guides.

## Weekly Search Console checks

- Performance: filter by `page` prefix, then compare clicks, impressions, CTR, and average position for `/en/`, `/ko/`, and `/tw/`.
- Country: check whether impressions are coming from expected regions, then compare outliers before changing page copy.
- Query: group queries by intent such as missing points, levels, promotions, subscriptions, gift cards, and country differences.
- Index coverage: confirm new international articles are indexed and no `noindex`, canonical, redirect, or duplicate warnings remain.

## Decision rules

- If a page gets impressions but low CTR, revise title and description first.
- If a page gets country impressions from an unexpected region, check language fit before adding more internal links.
- If a query cluster appears repeatedly, add one focused article rather than expanding unrelated pages.
- If index coverage shows a warning, fix crawlability before judging content quality.

## Minimal evidence to keep

- Date checked.
- Search Console filter used.
- Top country and query changes.
- Pages with index coverage warnings.
- Action taken or explicit decision to wait.
